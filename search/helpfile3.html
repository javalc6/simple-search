Tools

Before using tools you have to perform a search with specified options. Tools are applied only on the file that have met the specified conditions.

Copy files to folder

All the files that have met the specified search conditions are copied to the folder you specify.

Move files to folder

All the files that have met the specified search conditions are moved to the folder you specify.
The operation cannot be undone.

Delete files

All the files that have met the specified search conditions are deleted.
Warning: The operation cannot be undone.

Touch files

All the files that have met the specified search conditions are touched, i.e. the time/date of each file is updated to the current one.
The operation cannot be undone.

Replace files

The content of the files that met search criteria is updated by replacing a string with another one.
The operation cannot be undone, but the application create backup files with the extension .bak
