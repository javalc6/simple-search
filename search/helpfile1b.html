Commands

File Menu

Search: search for files using specific options.

Export filelist: export the list of files that met search criteria to a text file.

Exit: quits search application.

Tools Menu

Copy files to folder: copy files that met search criteria to folder specified by the user

Move files to folder: move files that met search criteria to folder specified by the user

Delete files: delete files that met search criteria

Touch files: touch files that met search criteria

Replace files: updates files that met search criteria by replacing a string with another one

Help menu

Help Topics: display the Help window.

About Search tools: displays the version number of Search tools
